The Compliance Nightmare Nobody Planned For

01

Customer PII in LLM Prompts

A support agent pastes a customer's full name, address, and account details into ChatGPT to draft a response. That data now lives on OpenAI's servers. Under GDPR, you just transferred personal data to a third-party processor without consent or a DPA.

02

Source Code in AI Assistants

Developers paste proprietary algorithms, database schemas, and API keys into coding assistants. Trade secrets disclosed. IP protections weakened. And you have zero visibility that it happened.

03

Financial Data Exposure

An analyst uploads quarterly projections to an AI for formatting. Material non-public information now sits in an external system. SOX and SEC reporting obligations just got complicated.

04

Health Records in AI Tools

Clinical staff use AI to summarize patient notes. PHI transferred to a non-BAA system. HIPAA violation. Potential fines from $100 to $50,000 per violation, up to $1.5M per year.

How PowerGrabber Protects You

Browser Extension Captures Traffic

Lightweight extension uses the webRequest API to observe every HTTP request. Works on Chrome, Firefox, Edge, Safari, Opera. No network proxy needed. No TLS interception.

Real-Time WebSocket Streaming

Events stream instantly to a local Rust backend over an authenticated WebSocket. 32-character token auth, 10-second handshake timeout, 500 events/sec rate limit.

Automatic Credential Redaction

Sensitive headers (Authorization, Cookie, API keys) are scrubbed before storage. URL parameters like tokens, passwords, and session IDs are sanitized. Your audit logs don't become a liability.

LLM API Detection

Automatically identifies calls to OpenAI, Anthropic, Google AI, Cohere, Mistral, HuggingFace, and more. Know exactly which AI services your organization is using — approved or not.

Structured Audit Output

JSONL output format for SIEM integration. Human-readable text mode for investigation. Every event includes timestamp, tab URL, request ID, and full header map.
