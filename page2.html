PowerGrabber

Browser-native LLM API traffic capture. Install a lightweight browser extension, run a Rust backend, and see every AI API call your browser makes — in real time.

langRust browsers6 supported latency<1ms securitytoken auth

Why?

LLMs are the new shadow IT. Developers paste source code into ChatGPT. Sales teams feed customer data to Claude. Analysts upload spreadsheets to Gemini. And nobody knows it's happening.

PowerGrabber gives you visibility. Not by proxying (which breaks TLS), not by parsing logs (which miss browser-side calls), but by capturing at the source — the browser's own network stack.

Core Capabilities

Traffic Capture

Every HTTP request and response. Full headers, URLs, status codes, timing. Five webRequest event types merged into a coherent event stream.

Automatic Redaction

Authorization, Cookie, Set-Cookie, API keys, CSRF tokens — 8 header types scrubbed. URL params like token, api_key, password — 14 patterns sanitized.

Cross-Browser

One extension codebase. MV3 for Chrome, Edge, Opera, Brave, Vivaldi, Arc. MV2 for Firefox. Safari via Xcode wrapper. Same backend for all.

Backpressure

try_send with drop counting. 1000-event offline buffer. Bounded tab cache. Periodic flush at 500ms. No memory leaks, no runaway queues.

Security Model

  • Token auth — 32-character random token, validated on first message within 10-second window
  • Rate limiting — 500 events/second per connection, excess silently dropped
  • Message size cap — 1MB max, prevents memory exhaustion
  • Connection limits — 16 concurrent clients max, RAII guard pattern
  • Input validation — URL length (8KB), header count (200), header value size (8KB), timestamp bounds
  • Self-traffic filtering — extension ignores its own WebSocket traffic

Use Cases

For Security Teams

Monitor which LLM APIs your organization's browsers are hitting. Detect unauthorized AI tool usage. Audit data sent to third-party AI services.

For Developers

Debug AI integrations. See exactly what payloads your app sends to OpenAI/Anthropic/Google. Inspect headers, timing, error codes in real time.

For Compliance

Prove to auditors that sensitive data isn't leaking to AI services. Automatic redaction means your monitoring logs themselves don't become a liability.
