Why Rust?

Zero-Cost Abstractions

Rust's ownership model means no garbage collector, no runtime overhead, no stop-the-world pauses. Every cycle goes to processing your events.

tokio Async Runtime

Non-blocking I/O for WebSocket connections. Each browser connection gets its own lightweight task. No thread-per-connection overhead.

Memory Safety

Compile-time guarantees against buffer overflows, use-after-free, and data races. Security-critical code that's safe by construction.

RAII Resource Management

Connection counting via Drop impl. Timers, buffers, and handles cleaned up deterministically. No leaked resources, ever.

Backpressure by Design

try_send on bounded channels. Drop counting instead of blocking. The system degrades gracefully under load — it never freezes.

Single Binary

Compile once, copy everywhere. No Python venv, no Node modules, no JVM. One file. Works on Linux, macOS, Windows.
