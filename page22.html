The OWASP Agentic Security Threats

ASI10 — Rogue Agents

Uncontrolled Autonomous Behaviour

AI agents that deviate from intended behaviour — contacting unexpected endpoints, escalating privileges, or exfiltrating data. Without monitoring, these actions are invisible until the damage is done.

ASI01 — Goal Hijacking

Prompt Injection Redirects Agent Goals

Malicious input (in emails, documents, web pages) causes the AI agent to abandon its original task and pursue an attacker-controlled objective — often data exfiltration disguised as normal tool use.

Real-World Attack: The EchoLeak Scenario

EchoLeak: Email-Triggered Silent Exfiltration

An attacker embeds a prompt injection payload in an email. When an AI assistant (like Microsoft Copilot) processes the email, the injected instructions cause it to silently exfiltrate sensitive data from the user's mailbox to an attacker-controlled endpoint.
