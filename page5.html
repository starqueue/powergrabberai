How It Works

Browser-Native Capture

A lightweight browser extension uses the webRequest API to observe every HTTP request. No proxy. No TLS interception. Works on Chrome, Firefox, Edge, Safari, and Opera.

LLM Endpoint Detection

Automatically identifies API calls to OpenAI, Anthropic, Google, Cohere, Mistral, HuggingFace, and custom endpoints. Know exactly which AI services are being consumed.

Real-Time WebSocket Stream

Events flow from browser to backend in real time over an authenticated WebSocket connection. Token-based auth, rate limiting at 500 events/sec, 1MB message cap.

Credential Redaction

API keys, auth headers, cookies, and session tokens are automatically scrubbed. You see the traffic patterns without the secrets. Your monitoring data stays clean.

JSONL Export

Structured event output for direct SIEM ingestion. Every event includes method, URL, headers, status, timing, tab context. Ready for your analytics pipeline.

Rust Performance

Sub-millisecond capture latency. Backpressure with try_send and drop counting. Bounded buffers everywhere. No memory leaks. Handles high-traffic browsers without breaking a sweat.
