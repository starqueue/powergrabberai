Why Open Source Matters for Security Tools

🔍

Full Auditability

A tool that monitors browser traffic must itself be trustworthy. Open source means anyone — your security team, external auditors, the community — can verify exactly what PowerGrabber does.

🔒

No Hidden Data Collection

Closed-source monitoring tools might phone home. PowerGrabber runs entirely local. WebSocket to 127.0.0.1. No telemetry, no analytics, no external calls. Verify it yourself.

🤝

Community Security

Security vulnerabilities are found faster when thousands of eyes can read the code. Every PR is public. Every security fix is documented. No silent patches.

The Complete Codebase

~1,200 lines of Rust + ~370 lines of JavaScript. Small enough to read in an afternoon.

src/
main.rs — CLI entry point, arg parsing, auth token generation
server.rs — WebSocket server, auth, rate limiting, RAII guards
protocol.rs — Extension message parsing, field validation
sanitize.rs — Header redaction, URL param sanitization
types.rs — Network event types, capture events, stats
output.rs — JSONL and text formatters
error.rs — Error types
lib.rs — Module exports
extension/
background.js — webRequest hooks, WebSocket client, buffering
manifest.json — Chrome/Edge/Safari (MV3)
manifest.firefox.json — Firefox (MV2)
popup/
popup.html — Status display, config UI
popup.js — Status polling, config save

Ways to Contribute

Add Redaction Rules

Know a sensitive header or URL parameter we missed? The sanitize module is designed for easy extension. Add a pattern, add a test.

New Output Formats

Need HAR output? SQLite storage? CSV export? The output module is pluggable. Write a formatter, register it in main.

Browser Testing

Verify the extension works on your browser. Report quirks with specific Chromium forks, Firefox versions, or Safari builds.

Security Audit

Found a vulnerability? The security analysis document covers 34 analysis techniques. Help us get to zero findings.

A security tool you can't audit is a liability, not an asset. PowerGrabber is small enough to read, transparent enough to trust, and open enough to improve.
