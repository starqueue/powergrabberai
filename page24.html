Token Delta Tracking

How token burn is computed

PowerGrabber records remaining_tokens from every response. The delta between consecutive responses reveals exact token consumption per request — without needing to parse request/response bodies.

Time remaining_tokens Delta Model Tool
14:02:01 720,000 — gpt-4o Cursor
14:02:18 716,480 -3,520 gpt-4o Cursor
14:02:34 714,210 -2,270 gpt-4o Cursor
14:02:51 706,890 -7,320 gpt-4o ChatGPT
14:03:09 703,100 -3,790 gpt-4o Cursor
Token delta formula: delta = remaining_tokens[n-1] - remaining_tokens[n]

Burn rate (tokens/minute): burn_rate = sum(deltas_last_5min) / 5

Projected exhaustion: time_remaining = remaining_tokens[latest] / burn_rate
