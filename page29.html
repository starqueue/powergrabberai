Budget Alerts Before the Invoice Arrives

Set per-provider, per-team, and global AI spend limits. Get notified at 50%, 75%, 90%, and 100% — in Slack, PagerDuty, or anywhere you need.

Configuration
Define budgets in plain TOML
One config file controls global limits, per-provider caps, and per-team budgets. No UI-clicking required.
# powergrabber-budget.toml

[global]
daily_limit   = 500.00
weekly_limit  = 3000.00
monthly_limit = 10000.00
currency      = "USD"

[alerts]
thresholds = [ 50, 75, 90, 100 ]

[providers.openai]
daily_limit  = 120.00
weekly_limit = 700.00

[providers.anthropic]
daily_limit  = 200.00
weekly_limit = 1200.00

[providers.google]
daily_limit  = 80.00

[teams.engineering]
daily_limit   = 150.00
monthly_limit = 3500.00

[teams.marketing]
daily_limit   = 30.00
monthly_limit = 600.00

[teams.research]
daily_limit   = 250.00
monthly_limit = 5000.00
Thresholds
Four-stage alert escalation
Each threshold triggers a distinct notification. You always know where spend stands before it becomes a problem.
50%
Heads Up
Informational ping. Half the budget used — no action needed yet.
75%
Warning
Three-quarters spent. Time to review usage patterns.
90%
Critical
Almost at limit. Escalate to budget owner immediately.
100%
Breached
Budget exceeded. Policy engine can auto-block or alert leadership.

Integrations
Webhook-first alert delivery
Every threshold breach fires a POST to your configured endpoints. Slack, PagerDuty, or your own internal tooling.

Slack Webhook

Post to any channel via incoming webhook URL.

[webhooks.slack] url = "https://hooks.slack.com/services/T.../B.../xxx" events = ["threshold_warning", "threshold_breach"] format = "slack"

PagerDuty

Create incidents on budget breach for on-call response.

[webhooks.pagerduty] url = "https://events.pagerduty.com/v2/enqueue" token = "${PAGERDUTY_TOKEN}" events = ["threshold_breach"] severity = "warning"

Custom Endpoint

POST JSON to any HTTP endpoint. Add custom headers, filter by event type.

[webhooks.custom] url = "https://internal.corp/api/ai-budget-alerts" method = "POST" headers = { Authorization = "Bearer ${INTERNAL_TOKEN}", Content-Type = "application/json" } events = ["threshold_warning", "threshold_breach", "daily_summary"]

"scope":     "team",

"threshold": 100,

"limit":     30.00,
"period":    "daily",
