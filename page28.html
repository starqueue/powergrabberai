// USAGE CLASSIFICATION

Not all AI usage is the same. The billing model, data exposure, and risk profile differ significantly.

🌐

Web UI Usage

Browser-based access via subscription (ChatGPT Plus, Perplexity Pro). Data enters via text input and file upload. Provider retains conversation history.

Subscription billing
⚙

API Usage

Programmatic access via API keys. Per-token billing. Data enters via HTTP POST. Provider may or may not retain data depending on terms and settings.

Per-token billing
💻

Coding Tool Usage

IDE extensions and CLI tools (Cursor, Copilot, Claude Code). Sends code context automatically. Mixed billing: some subscription, some per-token. Highest volume of sensitive data.

Mixed billing
