Why Honesty Builds Trust

Most compliance tools stretch thin signals into confident verdicts. PowerGrabber takes the opposite approach: if we cannot see inside request bodies, we will not pretend we can assess prompt injection (LLM01) or system prompt leakage (LLM07).

The 9 items we can assess are grounded in observable, repeatable HTTP metadata: rate-limit headers, response status codes, endpoint patterns, model version strings, authentication headers, and traffic flow graphs.

For auditors: A scorecard that says "NOT ASSESSABLE" for 11 items is more trustworthy than one that claims PASS for all 20. Your auditors will agree.
