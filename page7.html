[*] PowerGrabber v0.2.1 — Browser LLM Traffic Interceptor
[*] For security researchers, red teamers, and defenders
// ─── WHAT IS IT ──────────────────────────────────────────────
$ powergrabber --explain
PowerGrabber is a browser extension + Rust backend that captures
every HTTP request your browser makes to LLM API endpoints.
It uses the webRequest API — no proxy, no TLS interception,
no MITM. The extension sits inside the browser and watches.
[!] Think of it as tcpdump for AI API calls, but at the
application layer, with automatic credential redaction.
// ─── QUICK START ─────────────────────────────────────────────
$ cargo run -- --listen 127.0.0.1:9876

Load the extension. Paste the token. Start browsing.
Every AI API call will appear here in real time.

// ─── USE CASES ───────────────────────────────────────────────
$ powergrabber --list-scenarios
[01] Shadow AI Discovery
Find every LLM endpoint your org is calling. OpenAI, Anthropic, Google, Cohere, Mistral, HuggingFace — identified automatically.
[02] Data Exfiltration Detection
See what data leaves the browser to AI APIs. Capture request URLs and headers (with secrets redacted) for investigation.
[03] API Debugging
Real-time view of API calls, status codes, response headers. Like browser DevTools but streamed to your terminal.
[04] Compliance Monitoring
Generate audit trails of AI API usage. JSONL output feeds directly into your SIEM. Every event timestamped and attributed.
[05] Pentesting
Map an organization's AI attack surface. Discover unprotected endpoints, leaked API keys (before redaction), shadow integrations.
[06] CTF / Research
Capture and replay AI API traffic for analysis. Understand prompt injection patterns, token usage, rate limit behavior.
