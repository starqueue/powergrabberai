The Capture Pipeline

Intercept

webRequest API hooks into 5 browser events before they leave the network stack

Merge

Request + headers combined in a pending map with 500ms periodic flush

Stream

WebSocket push to local Rust backend. 1000-event buffer when offline

Validate

URL length, header count, timestamp bounds, request ID checks

Redact

8 header types + 14 URL param patterns sanitized before storage
